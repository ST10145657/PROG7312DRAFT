PROG7311_POE_Part3 – Inclusive Innovation Windows Forms Application

Overview

PROG7311_POE_Part3 is a Windows Forms Application developed as part of a semester project.
The purpose of the application is to provide an interactive main menu interface for users to:

Submit and view reports
View local events
Access available services
Change the application language (English, isiZulu, Afrikaans)
Exit the application safely
The project reflects the principles of inclusive innovation learned in Part 1 — emphasizing user participation, accessibility, and engagement through technology.

Project Summary

This project was built in three main phases:

Part 1 – Research & Planning

Studied the concept of inclusive innovation in South African municipalities.
Focused on user participation, collaboration, co-creation, and capacity building.
Highlighted how communities can take part in identifying needs and co-developing solutions.
Full document reference:
Hart, T.G.B., Booyens, I. & Sinyolo, S. (2019) Innovation for Development in South Africa.
Part 2 and 3 – Application Development Developed a Windows Forms system with the following key features:

Report Issues Form:
Users can log issues by entering a title, description, and category, and optionally attach files.

Local Events Form:
Displays upcoming community events.

Language Switcher:
Supports English, Afrikaans, and isiZulu for accessibility.

Custom User Interface: Clean layout, color consistency, and image-based buttons.

Exit Button:
Safely closes the application.
